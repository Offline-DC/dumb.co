## 🛠 Tech Intern

We're looking for a curious, motivated **Tech Intern** to support our engineering team across mobile, web, and backend projects, and to explore higher-level software and hardware solutions that empower our users to be **DUMB**.

This role is for someone with a basic understanding of programming fundamentals and a strong desire to learn new tools and languages. No prior job experience or formal education is required.

### Responsibilities

- Contribute to features, prototypes, and internal tools
- Read and write code across languages such as **Kotlin, SwiftUI, TypeScript, Bash, and React**
- Help test, debug, and document systems, with an eye toward quality and reliability
- Write and maintain tests to keep our shipping product stable
- Collaborate with cross-functional teams

### Requirements

- Understanding of core programming concepts in any language
- Care for code quality, testing, and thoughtful debugging
- Enthusiasm to learn new languages & software tools (experience with AI coding tools is a plus)
- Excited to tinker, think outside the box, and experiment
